## 七、答案与解析

### 基础概念答案
**问题1答案**: 
- 进程像工厂，线程像工人
- 线程轻量级原因：共享进程资源，创建开销小，切换成本低

**问题2答案**: B、D正确
- A错误：进程间通信复杂，线程间通信简单
- C错误：线程开销小，进程开销大

**问题3答案**: 线程不拥有独立的内存空间，共享进程的堆内存，只有独立的栈空间

---

### 编程题参考答案
**问题13参考答案**:
```java
public class SafeCounter {
    private int count = 0;
    
    public synchronized void increment() {
        count++;
    }
    
    public synchronized int getValue() {
        return count;
    }
}
```

**问题16参考答案**:
问题：count++不是原子操作，多线程环境下可能丢失更新
修复：使用synchronized或AtomicInteger
